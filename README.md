# UPS-Hat-for-Raspberry-Pi
<img src= "https://github.com/sbcshop/UPS-Hat-for-Raspberry-Pi/blob/main/Images/banner.png" />

## UPS Hat is a 5v power supply, It fit at the top of the raspberry pi, it gives power backup, while raspberry pi's USB power is cut off.it has various protection like short circuit protection, reverse battery protection and indicator, over charge/discharge protection, and over current protection.it has 5V USB output, convenient for powering other boards.
